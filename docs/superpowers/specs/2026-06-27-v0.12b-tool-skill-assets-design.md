# V0.12B Tool / Skill 资产库设计

## 决策

第一版资产库使用单表 `tool_skill_assets` 保存 Tool 与 Skill 两类资产。它是 Workspace 级资产，先支持创建和列表查询，不直接改 Agent 发布快照和 Runtime 执行。

## 字段

- `asset_type`：`tool` 或 `skill`。
- `name`：Workspace 内同类型唯一。
- `description`：资产说明。
- `parameter_schema`：JSON Schema，占位后续工具调用参数校验。
- `status`：`active` 或 `disabled`。
- `created_by`、`created_at`、`updated_at`：治理和审计基础。

## 权限

- 读取资产使用 `agent.read`。
- 创建资产使用 `agent.write`。
