- `src/pages/Artifacts.test.tsx`

## 步骤

1. 在页面测试中新增 RED：初始 URL 带 `artifactVersionId` 时自动打开详情。
2. 在页面测试中新增 RED：点击详情写入 query，关闭详情移除 query。
3. 在 `Artifacts.tsx` 使用 React Router search params 同步详情状态。
4. 跑 Artifact 页面测试，确认 RED 转 GREEN。
5. 更新实现说明和验收文档。
6. 跑相关前端回归、lint、build 和浏览器验证。
