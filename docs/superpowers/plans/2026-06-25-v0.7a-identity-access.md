# V0.7A Identity and Access Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为 ARC.ONE 增加本地账号登录、可撤销 Session、单组织多 Workspace、固定 RBAC、Reviewer 用户绑定和不可修改平台审计。

**Architecture:** FastAPI 继续作为单体 API，但认证、请求上下文、授权和审计拆成独立模块。所有业务资源通过显式 Workspace 路径访问，数据库查询强制限定 `workspace_id`。React 端增加 AuthProvider、WorkspaceRoute 和统一 API 客户端，前端权限状态只改善体验，后端 capability 校验是最终安全边界。

**Tech Stack:** FastAPI、SQLAlchemy 2、Pydantic、SQLite、Argon2id、React 19、React Router 7、TypeScript、Vitest、Testing Library、Playwright。

- [ ] **Step 4: Implement the security service**

Create `apps/api/app/security.py` with:

```python
import hashlib
import hmac
import secrets

from argon2 import PasswordHasher
from argon2.exceptions import InvalidHashError, VerifyMismatchError


class SecurityService:
    def __init__(self) -> None:
        self.password_hasher = PasswordHasher()

    def hash_password(self, password: str) -> str:
        return self.password_hasher.hash(password)

    def verify_password(self, password: str, encoded: str) -> bool:
        try:
            return self.password_hasher.verify(encoded, password)
        except (VerifyMismatchError, InvalidHashError):
            return False

    @staticmethod
    def new_token() -> str:
        return secrets.token_urlsafe(32)

    @staticmethod
    def digest_token(token: str) -> str:
        return hashlib.sha256(token.encode("utf-8")).hexdigest()

    @staticmethod
    def token_matches(token: str, digest: str) -> bool:
        return hmac.compare_digest(
            SecurityService.digest_token(token),
            digest,
        )
```

- [ ] **Step 3: Centralize HTTP requests**

Extend `http.ts`:

```typescript
export async function apiFetch(
  input: RequestInfo | URL,
  init: RequestInit = {},
): Promise<Response> {
  const method = (init.method ?? 'GET').toUpperCase()
  const headers = new Headers(init.headers)
  const csrfToken = document.cookie
    .split('; ')
    .find((entry) => entry.startsWith('arc_one_csrf='))
    ?.split('=')[1]
  if (!['GET', 'HEAD', 'OPTIONS'].includes(method) && csrfToken) {
    headers.set('X-CSRF-Token', decodeURIComponent(csrfToken))
  }
  return fetch(input, {
    ...init,
    credentials: 'same-origin',
    headers,
  })
}
```

Dispatch a single `auth-session-expired` browser event when a protected request returns 401.

interface AuthContextValue {

  status: 'loading' | 'authenticated' | 'anonymous'

interface WorkspaceContextValue {

- every `.scratch/v0.7a-identity-access/issues/*.md` acceptance item is checked;
- a password or token never appears in logs, API list responses, database plaintext or frontend
  storage;
- anonymous and cross-Workspace access tests fail safely;
- all role/capability combinations are covered;
- current-user Reviewer actions replace client-selected actors;
- audit records exist for success, failure and denied outcomes;
- the full completion gate has fresh passing evidence;
- the user has completed human acceptance.
