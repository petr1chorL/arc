# V0.10H Remediation Tasks Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Persist remediation queue items as Workspace-scoped tasks and let users update their status in the Evaluations page.

**Architecture:** Add a small FastAPI/SQLAlchemy resource under the existing evaluations router, then wire the existing React remediation queue to create and update those tasks. Keep scope limited to create/list/status update.

**Tech Stack:** FastAPI, SQLAlchemy, Pydantic, SQLite migration helper, React, TypeScript, Vitest, Testing Library.
