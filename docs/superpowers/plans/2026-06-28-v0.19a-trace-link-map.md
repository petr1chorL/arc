# V0.19A Trace 链路索引实施计划

## 目标

把运行详情里的 Trace / Span 证据聚合成一个可读的链路索引，让排障人员先看到“哪个 Span 上发生了什么”。

## 计划

1. [x] 建立本地 PRD、Issue 与状态文档。
2. [x] 在 `src/pages/Observability.test.tsx` 增加 Trace 链路索引 RED 测试。
3. [x] 在 `src/pages/Observability.tsx` 增加链路聚合函数与组件。
4. [x] 在运行详情中渲染 root 和节点 Span 卡片。
5. [x] 在 `src/index.css` 补充链路索引样式。
6. [x] 跑相关测试、全量测试、lint、build。
7. [x] 浏览器验收运行观测详情。
8. [x] 更新验收文档和当前实现说明。
