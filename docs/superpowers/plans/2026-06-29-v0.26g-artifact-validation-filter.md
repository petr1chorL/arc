## 步骤

1. 后端测试新增 RED：`schemaValidationStatus=failed` 只返回失败 Artifact。
2. 前端测试新增 RED：页面选择失败状态后，请求 URL 包含 `schemaValidationStatus=failed`。
3. 实现 API 查询参数、派生状态过滤和 limit 应用。
4. 实现前端 API 参数和页面筛选控件。
5. 跑后端聚焦测试、前端 Artifact 测试、lint、build 和浏览器验证。
6. 更新当前实现说明和验收记录。
