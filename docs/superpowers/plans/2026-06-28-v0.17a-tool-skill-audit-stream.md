# V0.17A Tool / Skill Audit Stream Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 为 Tool / Skill 资产提供可查询的审计流，聚合生命周期审计事件和运行时调用记录。

**Architecture:** 复用现有 `AuditEventRecord` 和 `ToolSkillAssetInvocationRecord`，新增只读 API 聚合两类记录并按 `createdAt` 倒序返回。权限沿用 `audit.read`。

**Tech Stack:** FastAPI, SQLAlchemy, Pydantic v2, pytest.
