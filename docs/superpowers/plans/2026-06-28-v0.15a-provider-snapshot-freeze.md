# V0.15A Provider Snapshot Freeze Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:test-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Freeze Provider secret reference labels into published Agent versions and run published versions from their immutable snapshot.

**Architecture:** Keep the Provider asset mutable for drafts, but publish a non-secret `modelSecretRef` label into `AgentVersionRecord.snapshot`. `ExecutionService` should prefer this snapshot value and only fall back to the current Provider record for legacy snapshots created before V0.15A.

**Tech Stack:** FastAPI, SQLAlchemy, Pydantic JSON snapshots, pytest.
