# V0.8E Failure Classification Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在运行观测中为异常运行提供结构化失败原因、排障建议和前端筛选。

**Architecture:** 后端用确定性规则从 `WorkflowRun` 状态、错误文本和当前节点分类，不引入外部模型或观测栈。前端复用现有观测页的 URL query 筛选，把失败原因展示到风险列表、最近运行和详情处理建议中。

**Tech Stack:** FastAPI, Pydantic, SQLAlchemy, Pytest, React, TypeScript, Vitest, Testing Library, Vite.
