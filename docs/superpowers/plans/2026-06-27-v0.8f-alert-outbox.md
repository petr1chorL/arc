# V0.8F Alert Outbox Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 在运行观测中心提供轻量告警 Outbox，让异常运行和已有 SLA 通知可以在页面内统一查看。

**Architecture:** 不引入外部通知服务，不改 `notification_outbox` 表结构。后端在观测概览中返回 `alerts`：一部分由运行风险确定性投影生成，一部分来自已有 `NotificationOutboxRecord`。前端在观测页展示待处理告警列表，作为后续飞书、邮件、Webhook 消费器的输入视图。

**Tech Stack:** FastAPI, Pydantic, SQLAlchemy, Pytest, React, TypeScript, Vitest, Testing Library, CSS.
