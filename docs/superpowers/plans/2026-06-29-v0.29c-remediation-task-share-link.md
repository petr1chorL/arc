# V0.29C Remediation Task Share Link Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a one-click share link action to Remediation Task detail so collaborators can reopen the same task context.

**Architecture:** React builds the canonical task detail URL from the current origin and `workspacePath("evaluations?taskId=...")`, writes it to `navigator.clipboard`, and renders local success/failure feedback. No backend, persistence, audit, or routing change is needed.

**Tech Stack:** React, TypeScript, React Router, Vitest, Testing Library, Playwright browser smoke check.
