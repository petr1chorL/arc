Inside `remediationTaskDetail`, after source links, render buttons with the same disabled conditions as the card:

```tsx
<button ... onClick={() => void updateTaskStatus(highlightedRemediationTask, 'in_progress')}>
  标记处理中

<button ... onClick={() => void updateTaskStatus(highlightedRemediationTask, 'done')}>
  标记完成
</button>
```
