# ARC.ONE Soft UI 重设计实施计划

> **执行要求：** 使用 Superpowers 流程逐项实施，并在完成前运行完整验证。

**目标：** 将现有六个页面升级为已确认的平衡触感 Soft UI，同时保持全部功能与信息结构。

**架构：** 保留 React 组件、路由、模拟数据和 React Flow 行为。通过全局 Design Token、应用外壳少量语义调整和集中 CSS 覆盖完成视觉升级。

**技术栈：** React 19、TypeScript、Vite、React Router、React Flow、Lucide React、原生 CSS。
