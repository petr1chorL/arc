# V0.9H Golden Set Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 将 V0.9G 的临时批量回归样本升级为可持久化、可选择、可复用的 Golden Set / 回归样本集。

**Architecture:** 后端在 workspace 维度新增 `regression_sample_sets` 与 `regression_samples` 两类资产；前端评估中心读取样本集并把选中的 active samples 接入现有 Rubric evaluate API。

**Tech Stack:** FastAPI, SQLAlchemy, Pydantic, SQLite, React, TypeScript, Vitest, Testing Library。

---

## Scope

- 新增样本集与样本数据模型。
- 新增样本集列表、创建、样本新增接口。
- 新增评估中心样本集面板。
- 批量回归支持选择样本集运行。
- 补充验收文档和验证证据。

## Out of Scope

- 定时回归任务。
- 独立 Regression Run 历史表。
- 真实 LLM-as-Judge。
- 样本导入、导出、版本对比。
