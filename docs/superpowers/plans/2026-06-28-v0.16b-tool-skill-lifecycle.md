# V0.16B Tool Skill Lifecycle Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:test-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add edit, deactivate, and impact visibility for Tool / Skill assets.

**Architecture:** Extend the existing workspace `/asset-library` FastAPI surface with update, deactivate, and impact endpoints. Reuse Agent draft records and AgentVersion snapshots to compute impact. Extend the existing React AssetLibrary page with inline edit controls and impact cards.

**Tech Stack:** FastAPI, SQLAlchemy, Pydantic, React, TypeScript, Vitest, Testing Library.
