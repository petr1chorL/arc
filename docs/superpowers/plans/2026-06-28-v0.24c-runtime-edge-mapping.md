# V0.24C Runtime Edge Mapping Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Apply workflow edge field mappings during execution so downstream node input is generated from upstream JSON output.

**Architecture:** Add small pure helpers in `apps/api/app/execution.py` to parse simple object paths, extract values from upstream JSON outputs, and build mapped downstream JSON input. `ExecutionService.execute_workflow_from` uses the helper before falling back to the existing upstream-output concatenation behavior.

**Tech Stack:** FastAPI execution services, SQLAlchemy persisted NodeRun records, Python JSON, Pytest, Fake ModelGateway.
