# V0.18D 权限操作二次确认实施计划

## 目标

为成员与权限页的高风险权限操作补充保存前确认，降低误授权、误降权和误停用风险。

## 计划

1. [x] 建立本地 PRD、Issue 与状态文档。
2. [x] 在 `src/pages/Members.test.tsx` 增加高风险角色保存和停用操作 RED 测试。
3. [x] 在 `src/pages/Members.tsx` 增加确认弹窗状态与执行分流。
4. [x] 高风险角色变更确认后才调用 `updateMemberRole`。
5. [x] 停用 Membership / User 确认后才调用对应 API。
6. [x] 补充确认弹窗样式。
7. [x] 跑测试、lint、build。
8. [x] 浏览器验收。
9. [x] 更新验收文档和当前实现说明。
