# V0.28C Remediation Source Links Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add source navigation links on Remediation Task cards so task handlers can jump back to the originating Artifact and Workflow Run trace.

**Architecture:** Reuse existing RemediationTask data. Parse only stable `artifact:<artifactVersionId>` cluster keys for Artifact links, and use `sourceRunId` for Observability links. Keep the change front-end only and preserve existing task status/comment/retest behavior.

**Tech Stack:** React, React Router, TypeScript, Vitest, Testing Library.

## Task 2: Implement minimal source links

- [x] **Step 1: Update imports and workspace helper**

In `src/pages/Evaluations.tsx`, change the router import to:

```ts
import { Link, useSearchParams } from 'react-router-dom'
```

Change workspace destructuring from:

```ts
const { workspace } = useWorkspace()
```

to:

```ts
const { workspace, workspacePath } = useWorkspace()
```

- [x] **Step 2: Add a small artifact-key helper**

Near other pure helpers in `src/pages/Evaluations.tsx`, add:

```ts
function getArtifactVersionIdFromTask(task: RemediationTask) {
  const prefix = 'artifact:'
  return task.clusterKey.startsWith(prefix) ? task.clusterKey.slice(prefix.length) : ''
}
```

- [x] **Step 3: Add links in the task card action row**

Inside the `remediationTasks.map((task) => { ... })` block, derive:

```ts
const artifactVersionId = getArtifactVersionIdFromTask(task)
const artifactParams = new URLSearchParams({ artifactVersionId })
const traceParams = new URLSearchParams({ runId: task.sourceRunId })
```

Inside `.remediation-task-actions`, before status buttons, render:

```tsx
{artifactVersionId && (
  <Link
    aria-label={`查看 ${task.id} 产出物`}
    className="button secondary small"
    to={workspacePath(`artifacts?${artifactParams.toString()}`)}
  >
    <FileText size={14} />
    查看产出物
  </Link>
)}
<Link
  aria-label={`查看 ${task.id} 运行链路`}
  className="button secondary small"
  to={workspacePath(`observability?${traceParams.toString()}`)}
>
  <ArrowRight size={14} />
  查看运行链路
</Link>
```
