# V0.16E Runtime Stable Tool References Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Agent 运行时优先按发布快照里的稳定 Tool 资产 ID 调用 HTTP Tool，避免 Tool 改名破坏已发布版本执行。

**Architecture:** `WorkflowExecutionEngine` 从 AgentVersion snapshot 读取 `toolAssetRefs`，将 Tool 引用传入 `_invoke_bound_http_tools`。该方法优先按资产 ID 查询 active HTTP Tool，旧快照没有引用时继续按名称数组兜底。

**Tech Stack:** FastAPI, SQLAlchemy, SQLite JSON snapshot, pytest, React TypeScript.
