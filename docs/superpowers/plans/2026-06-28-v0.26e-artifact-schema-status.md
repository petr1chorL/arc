# V0.26E Artifact Schema 轻量校验状态实施计划

## 范围

给 Artifact 实例页增加前端轻量 Schema 校验状态，帮助用户快速判断 Artifact 内容是否满足绑定 Data Object Snapshot 的基础对象契约。

## 文件

- `src/pages/Artifacts.test.tsx`
- `src/pages/Artifacts.tsx`
- `src/index.css`
- `docs/CURRENT_IMPLEMENTATION.md`
- `docs/ACCEPTANCE_V0.26E.md`

## 步骤

1. 在 `src/pages/Artifacts.test.tsx` 增加红灯测试，覆盖通过状态、失败状态和详情失败原因。
2. 运行聚焦测试，确认测试因目标行为缺失而失败。
3. 在 `src/pages/Artifacts.tsx` 增加轻量校验 helper，并把状态展示到列表和详情弹窗。
4. 在 `src/index.css` 增加状态标识与失败原因样式。
5. 运行聚焦测试和相关 Artifact 测试，确认回归通过。
6. 更新当前实现说明和 V0.26E 验收文档。
7. 运行 `npm run lint`、`npm run build`，并完成浏览器路径验证。

## 验证命令

```powershell
npm run test -- src/pages/Artifacts.test.tsx --run

```
