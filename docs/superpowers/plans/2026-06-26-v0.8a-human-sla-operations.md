> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a Human Task SLA operations slice to the existing observability page.

**Architecture:** Reuse the existing HumanTaskService SLA refresh logic and expose a read-only Workspace-scoped observability endpoint. The frontend extends the observability API client and page with an SLA backlog panel that links to the existing human review workbench.

**Tech Stack:** FastAPI, SQLAlchemy, Pydantic, Pytest, React, TypeScript, Vitest, Testing Library, native CSS.
