# V0.29D Remediation Task Detail Close Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a clear close action to Remediation Task detail so users can leave a shared/deep-linked task context and return to the task list.

**Architecture:** React removes `taskId` from `useSearchParams`, keeps the Evaluations page mounted, and hides the detail region because no task is highlighted. No backend, persistence, audit, or routing change is needed.

**Tech Stack:** React, TypeScript, React Router, Vitest, Testing Library, Playwright browser smoke check.
