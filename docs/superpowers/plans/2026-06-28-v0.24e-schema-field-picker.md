# V0.24E Schema 字段选择器实施计划

## 目标

让工作流连线映射面板能从当前工作流 `inputSchema` 提供字段快捷选择，减少手写 JSONPath 的出错概率。

## 范围

- [x] 从简单对象型 `inputSchema.properties` 派生字段选项。
- [x] 在连线映射面板增加源字段快捷选择，写入 `$.<field>`。
- [x] 在连线映射面板增加目标字段快捷选择，写入 `$.input.<field>`。
- [x] 保留自由文本路径输入。
- [x] 不改变后端 API 与 `edge.data.mappings` 数据结构。

## 不做

- [x] 不实现嵌套 Schema 字段树。
- [x] 不实现完整 JSONPath Builder。
- [x] 不引入 Data Object 或节点输出 Schema。
- [x] 不改运行时映射引擎。
