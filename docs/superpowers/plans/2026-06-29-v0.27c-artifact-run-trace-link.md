# V0.27C Artifact 运行链路入口实施计划

## 范围

让 Artifact 详情弹窗可以跳转到对应 Run 的 Observability 链路页。

## 文件

- `src/pages/Artifacts.test.tsx`
- `src/pages/Artifacts.tsx`
- `docs/CURRENT_IMPLEMENTATION.md`
- `docs/ACCEPTANCE_V0.27C.md`

## 步骤

1. 新增 RED：打开 Artifact 详情后，应出现“查看运行链路”链接。
2. 新增 RED：链接应指向当前 Workspace 的 `observability?runId=<runId>`，并编码 `runId`。
3. 在 `Artifacts.tsx` 中使用 `workspacePath` 生成链接，并放入详情弹窗头部动作区。
4. 跑 Artifact 页面测试、相关回归、lint、build 和浏览器验证。
5. 更新当前实现说明和验收文档。
