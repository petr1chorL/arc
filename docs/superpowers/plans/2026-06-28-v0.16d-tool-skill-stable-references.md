# V0.16D Tool / Skill Stable References Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Agent 绑定 Tool / Skill 时保存稳定资产引用，并在发布版本中冻结资产快照，避免资产改名导致影响面丢失。

**Architecture:** 保留现有 `tools` / `skills` 名称数组作为兼容层；新增 `tool_asset_refs` / `skill_asset_refs` 作为稳定资产引用。影响面查询优先按资产 ID 匹配，旧数据按名称兜底。

**Tech Stack:** FastAPI, SQLAlchemy, SQLite JSON columns, Pydantic v2, pytest, Vitest, React TypeScript.
