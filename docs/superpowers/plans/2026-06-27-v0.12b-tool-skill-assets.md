# V0.12B Tool / Skill Assets Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:test-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Add a backend Workspace-level Tool / Skill asset library with create and list APIs.

**Architecture:** Store tools and skills in one SQLAlchemy table keyed by Workspace, type, and name. Expose API endpoints in the existing workspace router surface in `apps/api/app/main.py`.

**Tech Stack:** FastAPI, Pydantic, SQLAlchemy, Pytest.
