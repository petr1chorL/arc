# ARC.ONE 落地任务计划

> 基线：2026-06-25，当前为 V0.6 可运行原型，不等同于生产平台。

## 当前目标

围绕一条真实业务试点，把现有 Agent、工作流、运行和人工审核原型，
逐步建设为可安全使用、可评价、可观测、可部署的 V1.0 企业生产版。
