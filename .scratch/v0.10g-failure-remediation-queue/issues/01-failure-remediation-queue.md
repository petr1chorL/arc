# Issue 01：失败原因修复队列

Category: enhancement
Status: done
PRD: `../PRD.md`

## 建设内容

在评估中心的 `Failure Pattern Summary` 下方，为当前最新 Regression Run 的失败原因组生成可读的修复队列。

## 验收标准

- [ ] 最新 Run 存在失败原因组时，页面展示 `Failure Remediation Queue`。
- [ ] 每个修复项展示优先级、原因标题、样本数、最低分和代表样本 ID。
- [ ] 优先级按失败样本数和最低分确定，并按优先级排序。
- [ ] 每个修复项展示明确的建议动作和复测提示。
- [ ] 没有失败原因组时，不展示修复队列。
- [ ] 原有趋势、洞察、失败聚类、Run 详情和 Run 对比功能不回归。

## 前置依赖

V0.10F 失败样本聚类与原因摘要。
